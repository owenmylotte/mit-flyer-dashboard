one-time setup
  1. Open your Google Sheet with the flyer tracking data.
  2. Make sure the sheet has columns: Building, Bldg #, Floor, Spot Name, Spot Type, Assigned To, Posted, Date Posted, Notes (in that order).
  3. Go to File → Share → Publish to web.
  4. Select the Coverage Data sheet (not "Entire Document").
  5. Choose CSV as the format, then click Publish.
  6. Copy the URL it gives you and paste it below.
The URL should contain output=csv at the end. This is read-only — viewers of your dashboard cannot edit the spreadsheet. Only people you've shared the sheet with directly can edit it.
